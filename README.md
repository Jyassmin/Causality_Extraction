# **"NLP를 사용한 COPD 인과관계 추출 및 네트워크 시각화"**
<br><br>

🏆 2022 제 12회 숭실 캡스톤디자인 경진대회

# **프로젝트 소개**
- 본 프로젝트는 의학저널로부터 얻은 만성폐쇄성폐질환(COPD) 관련자료를 바탕으로 인과관계추출을 통해 COPD의 발병 원인 또는 연관성을 가지는 단어를 파악하고자 한다.
